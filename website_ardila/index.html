Tentang Website Ini

Website ini saya buat menggunakan HTML dan CSS murni tanpa framework apapun. Ini adalah pertama kalinya saya membuat website multi-halaman yang punya navigasi tersambung, layout yang rapi, dan desain yang cukup enak dilihat. Lumayan bangga juga sih hasilnya, hehe.

Di sini kamu bisa menemukan berbagai informasi tentang saya — mulai dari biodata lengkap di halaman Profil, koleksi foto di Galeri, sampai artikel yang saya tulis sendiri tentang dunia pemrograman di halaman Artikel. Kalau mau ngobrol atau sekadar kenalan, langsung aja ke halaman Kontak ya.
